# MUSICYOS - Plateforme de Création Musicale

Bienvenue sur **MUSICYOS**, une plateforme web innovative dédiée à la **création et production musicale**. Accessible, intuitive et puissante.

## 🎵 Caractéristiques Principales

### 1. **Console de Mixage Interactive**
- 6 potentiomètres clignotants avec animation fluide
- Contrôle du volume, reverb, delay, basses, médiums et aigus
- Interface professionnelle avec feedback visuel en temps réel

### 2. **Moniteurs Audio Avancés**
- Visualiseur de forme d'onde stéréo (gauche et droite)
- Analyseur de spectre en temps réel
- Mise à jour dynamique synchronisée avec l'audio

### 3. **Sélecteur de Styles Musicaux**
- Électronique
- Hip-Hop / Rap
- Ambient
- Classique
- Jazz
- Expérimental

### 4. **Dépôt d'Échantillons Vocaux**
- Upload drag-and-drop
- Support de multiples formats (MP3, WAV, OGG, M4A)
- Limite de 50MB par fichier
- Barre de progression en temps réel

### 5. **Formulaire de Commande/Contact**
- Informations personnelles
- Sélection du sujet et du style musical
- Estimation de budget
- Message détaillé du projet

### 6. **Démo Interactive**
- Presets prédéfinis (Chillwave, Techno, Ambient, Jazz)
- Contrôle du tempo (60-180 BPM)
- Ajustement du volume
- Visualiseur canvas animé

### 7. **À Propos**
- Présentation du créateur
- Raisons d'être du projet
- Vision et valeurs

## 🚀 Démarrage Rapide

1. **Clonez le repository** :
   ```bash
   git clone https://github.com/TKIFF2026/MUSIC.git
   cd MUSIC
   ```

2. **Ouvrez le fichier `index.html`** dans votre navigateur :
   ```bash
   # Vous pouvez utiliser un serveur local comme Live Server
   # ou simplement double-cliquer sur index.html
   ```

3. **Explorez les fonctionnalités** :
   - Testez la console de mixage
   - Écoutez la démo interactive
   - Chargez un fichier audio
   - Remplissez le formulaire de contact

## 🎨 Technologies Utilisées

- **HTML5** - Structure sémantique
- **CSS3** - Animations et gradients modernes
- **JavaScript (Vanilla)** - Web Audio API, DOM Manipulation
- **Web Audio API** - Synthèse et analyse audio
- **Canvas API** - Visualiseurs graphiques

## 🔊 Fonctionnalités Audio
